# Machine Learning Aplicado à Saúde — Diabetes

Este notebook demonstra a aplicação de **Machine Learning** na área da saúde, utilizando o dataset de **Diabetes** disponível na biblioteca `scikit-learn`.

O projeto realiza:
-  **Análise Exploratória de Dados (EDA)**
-  **Aprendizagem Supervisionada** (Lasso Regression)
-  **Aprendizagem Não Supervisionada** (K-Means)
-  **Visualização de Resultados e Métricas**
